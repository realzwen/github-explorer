# 🔍 GitHub Repo Keşfet

GitHub'daki public repoları arama yapmadan kategori, dil ve sıralama filtreleriyle keşfet. İstersen yazarak da arama yapabilirsin.

**Demo:** [github.com/realzwen/github-explorer](https://github.com/realzwen/github-explorer/tree/main)

---

## Özellikler

- Kategori filtreleri: Web, API, CLI, Database, DevOps, AI/ML, Mobile, Security
- Dil filtresi: JavaScript, TypeScript, Python, Go, Rust
- Sıralama: Yıldız, Fork, Son güncelleme
- Yazarak arama desteği
- Sayfalama
- Tamamen client-side, backend yok

## Kullanım

```bash
git clone https://github.com/realzwen/github-explorer.git
cd github-explorer
```

`index.html` dosyasını tarayıcıda aç, hazır.

## Teknoloji

Vanilla HTML / CSS / JavaScript — GitHub REST API v3
